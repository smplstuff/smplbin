about smplbin

smplbin is a simple, no-frills file sharing service designed for quick and easy file transfers.

our philosophy is simplicity: no registration required, no tracking, and a straightforward user experience. just upload, share, and go.

key features

  • upload up to 5MB per file
  • share multiple files at once
  • files automatically expire after 3 days
  • no account needed
  • simple rest api available

smplbin was created in 2025 as a lightweight alternative to complex file sharing services.
